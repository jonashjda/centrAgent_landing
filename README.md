# CentrAgent Landing Page

This is the landing page for CentrAgent, designed to be hosted on a separate domain.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Configuration

Update the `platformUrl` in `src/App.jsx` to point to your platform's URL (e.g., `https://app.yourdomain.com`).
